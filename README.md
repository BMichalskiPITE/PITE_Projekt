# UberTravel

# prefered requirements:
 - python: 3.5.2
 - npm: 5.6.0
 - Angular CLI: 1.5.5
 - Node: 9.2.1
 - OS: linux x64


# commands:
 - run server
```
python3 server/uberTravel/manage.py runserver
```
 - install client modules
```
cd serve/
npm install
```
 - run client
```
cd server/
ng serve
```

# default ports:
 - client - localhost:4200
 - server - localhsot:8000
